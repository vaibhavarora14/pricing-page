# Modal

## API

| Name               | Description                                          |
| ------------------ | ---------------------------------------------------- |
| show: boolean      | shows/hides the modal.                               |
| title?: string     | Title of modal will be shown at top of modal.        |
| onClose?: function | function which will be trigged when modal is closed. |

## Example

```js
<Modal title="My Modal"></Modal>
```
